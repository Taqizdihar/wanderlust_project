Apa Itu Theme Web?

Theme atau tema web adalah tampilan dan desain dari sebuah website. Ibaratnya seperti "baju" yang dikenakan oleh situs tersebut. Tema ini mengatur warna, font, tata letak, dan elemen visual lainnya agar website terlihat menarik dan sesuai dengan tujuan atau branding yang diinginkan.

Misalnya, kalau kamu menggunakan WordPress, ada banyak tema yang bisa dipilih, mulai dari yang simpel hingga yang penuh animasi. Dengan mengganti tema, kamu bisa mengubah tampilan website tanpa harus menyentuh kode programnya.

Apa Itu View dalam Pemrograman Web?

View dalam pemrograman web berhubungan dengan bagaimana sebuah halaman web ditampilkan kepada pengguna. Kalau diibaratkan, view itu seperti layar yang kamu lihat di website, tempat di mana data atau informasi yang sudah diolah ditampilkan dengan tampilan yang menarik.

Dalam konsep Model-View-Controller (MVC)—yang sering dipakai dalam framework seperti Laravel (PHP) atau Django (Python)—view berperan sebagai bagian yang menangani tampilan. Jadi, misalnya ada data produk dari database, view-lah yang akan menampilkan daftar produk tersebut dalam bentuk yang bisa dipahami pengguna, seperti tabel atau kartu produk.

  • Theme → Mengatur tampilan keseluruhan website (desain, warna, font, dll.).
  • View → Bertanggung jawab untuk menampilkan data di website dalam pemrograman web (khususnya dalam arsitektur MVC).

Cara Menggunakan Tampilan (HTML/CSS) dari Theme di PHP

Kalau kamu punya theme berbasis HTML & CSS dan ingin menggunakannya di PHP, ada beberapa langkah yang bisa kamu ikuti supaya tampilannya tetap rapi dan lebih mudah dikelola.

1. Siapkan Theme yang Mau Dipakai

  • Kalau kamu download theme dari internet (misalnya theme HTML gratis), biasanya file-nya dalam format .zip yang harus diekstrak dulu.
  • Di dalamnya bakal ada folder yang berisi file HTML, CSS, gambar, dan mungkin juga JavaScript.
  • Kalau kamu bikin theme sendiri, pastikan semua file ini sudah siap.

2. Pisahkan Bagian yang Bisa Dipakai Berulang (Header & Footer)

Agar lebih mudah dikelola, sebaiknya pisahkan bagian yang sering muncul di banyak halaman, seperti header (menu navigasi) dan footer (bagian bawah halaman).

Misalnya, kita bisa buat:

  • header.php → Berisi bagian atas halaman (menu, logo, dll.)
  • footer.php → Berisi bagian bawah halaman (copyright, link, dll.)
  • index.php → Halaman utama

✍ Contoh header.php:

php
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Website Saya</title>
    <link rel="stylesheet" href="assets/css/style.css">
</head>
<body>
    <nav>
        <ul>
            <li><a href="index.php">Home</a></li>
            <li><a href="about.php">Tentang</a></li>
            <li><a href="contact.php">Kontak</a></li>
        </ul>
    </nav>

✍ Contoh footer.php:

php
    <footer>
        <p>© 2024 Website Saya. All Rights Reserved.</p>
    </footer>
</body>
</html>

3. Gunakan include() atau require() di Halaman PHP

Setelah dipisah, kita tinggal gabungkan di halaman utama menggunakan include() atau require().

✍ Contoh index.php:

php
<?php include('header.php'); ?>
    <h1>Selamat Datang di Website Saya</h1>
    <p>Ini adalah halaman utama.</p>
<?php include('footer.php'); ?>

Dengan cara ini, kalau nanti ada perubahan di header atau footer, kamu cukup edit satu file saja, tanpa harus ubah di semua halaman!

4. Hubungkan dengan Database (Opsional)

Kalau website kamu pakai database, kamu bisa mengambil data dari MySQL dan menampilkannya di theme.

✍ Contoh menampilkan daftar produk dari database:

php
<?php
include 'koneksi.php'; // File koneksi database
$query = mysqli_query($conn, "SELECT * FROM produk");
while ($row = mysqli_fetch_assoc($query)) {
    echo "<h2>" . $row['nama_produk'] . "</h2>";
    echo "<p>Harga: Rp" . $row['harga'] . "</p>";
}
?>

Dengan cara ini, kamu nggak perlu bikin halaman statis satu per satu, karena semua data bisa langsung diambil dari database.

5. Pastikan CSS & JavaScript Berjalan dengan Baik

Kadang-kadang, file CSS atau JavaScript nggak langsung jalan karena jalurnya salah. Pastikan semua file dipanggil dengan benar.

✍ Contoh di header.php:

html
<link rel="stylesheet" href="assets/css/style.css">
<script src="assets/js/script.js"></script>

Cara Menyusun Folder untuk Menyimpan File Tampilan (Theme) di PHP

Agar proyek PHP lebih rapi dan mudah dikelola, penting untuk menyusun folder dengan struktur yang jelas. Berikut adalah struktur folder yang sering digunakan untuk menyimpan file tampilan/theme di PHP:

bash
/mywebsite          → (Folder utama proyek)
│── /assets        → (Folder untuk file statis seperti CSS, JS, dan gambar)
│   │── /css       → (Folder untuk file CSS)
│   │   ├── style.css
│   │   ├── responsive.css
│   │── /js        → (Folder untuk file JavaScript)
│   │   ├── script.js
│   │── /images    → (Folder untuk gambar)
│── /includes      → (Folder untuk menyimpan file yang dipanggil di berbagai halaman)
│   │── header.php → (File header yang dipakai di semua halaman)
│   │── footer.php → (File footer yang dipakai di semua halaman)
│── /pages         → (Folder untuk halaman-halaman tambahan)
│   │── about.php  → (Halaman "Tentang Kami")
│   │── contact.php → (Halaman "Kontak")
│── /config        → (Folder untuk file konfigurasi)
│   │── database.php → (File koneksi ke database)
│── index.php      → (Halaman utama)
│── style.css      → (File CSS utama jika tidak menggunakan folder assets)
│── .htaccess      → (Konfigurasi server, opsional)

Penjelasan Struktur Folder

  1. /assets → Untuk menyimpan file statis seperti CSS, JavaScript, dan gambar.
  2. /includes → Tempat menyimpan file header dan footer, supaya bisa dipanggil di berbagai halaman dengan include().
  3. /pages → Untuk halaman-halaman tambahan seperti about.php atau contact.php agar tidak bercampur dengan file utama.
  4. /config → Folder untuk menyimpan file konfigurasi, misalnya koneksi database.
  5. index.php → Halaman utama website yang akan memanggil tampilan dari folder lain.

Contoh Pemanggilan File dalam PHP

✍ index.php

php
<?php include 'includes/header.php'; ?>
<h1>Selamat Datang di Website Saya</h1>
<p>Ini adalah halaman utama.</p>
<?php include 'includes/footer.php'; ?>

✍ header.php

php
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Website Saya</title>
    <link rel="stylesheet" href="assets/css/style.css">
</head>
<body>
    <nav>
        <ul>
            <li><a href="index.php">Home</a></li>
            <li><a href="pages/about.php">Tentang</a></li>
            <li><a href="pages/contact.php">Kontak</a></li>
        </ul>
    </nav>

✍ footer.php

php
    <footer>
        <p>© 2024 Website Saya. All Rights Reserved.</p>
    </footer>
</body>
</html>

│   │── contact.php → (Halaman "Kontak")
│── /config       → (Folder untuk konfigurasi, misalnya koneksi database)
│   │── database.php → (File koneksi ke database)
│── index.php     → (Halaman utama)
│── .htaccess     → (File konfigurasi server, opsional)

Penjelasan Struktur Folder

💡 /assets → Tempat file statis seperti CSS, JavaScript, dan gambar. Ini bikin tampilan website tetap keren.

💡 /includes → Buat file yang sering dipakai berulang, seperti header dan footer, supaya nggak perlu nulis ulang di setiap halaman.

💡 /pages → Menyimpan halaman tambahan seperti about.php dan contact.php, biar nggak tercampur dengan file utama.

💡 /config → Untuk file konfigurasi penting, misalnya koneksi ke database.

💡 index.php → Halaman utama website yang nantinya akan memanggil file header, footer, dan lainnya.

Contoh Penggunaan di PHP

🔹 index.php – Halaman utama website

php
<?php include 'includes/header.php'; ?>
<h1>Selamat Datang di Website Saya</h1>
<p>Ini adalah halaman utama.</p>
<?php include 'includes/footer.php'; ?>

🔹 header.php – Bagian atas website

php
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Website Saya</title>
    <link rel="stylesheet" href="assets/css/style.css">
</head>
<body>
    <nav>
        <ul>
            <li><a href="index.php">Home</a></li>
            <li><a href="pages/about.php">Tentang</a></li>
            <li><a href="pages/contact.php">Kontak</a></li>
        </ul>
    </nav>

🔹 footer.php – Bagian bawah website

php
    <footer>
        <p>© 2024 Website Saya. All Rights Reserved.</p>
    </footer>
</body>
</html>

Dengan begini, kalau nanti mau ubah tampilan menu atau footer, cukup edit di satu tempat aja, nggak perlu repot ubah di semua halaman!
